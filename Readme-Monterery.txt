About the POPFile installer for macOS 12.x (Monterery) or 13.x (Ventura)

***NOTE***

  This is the installer package of POPFile for macOS 12.x (Monterery) or 13.x (Ventura).
  If you are using macOS 11 or earlier, please download the BigSur or earlier version from here:

  http://getpopfile.org/download

Install POPFile

  Open 'POPFile-x.x.x-monterery.pkg' and follow the wizard.


Start to use POPFile

  Please see the Quick Start Guide on our web site:

  http://getpopfile.org/docs/quickstart


Uninstall POPFile

  Use 'uninstall-popfile.app' in the 'utilities' folder.

  Note: The uninstaller will remove POPFile program and data files.


For more information about POPFile for Mac OS X

  Please see: http://getpopfile.org/docs/HowTos:MacOSX
